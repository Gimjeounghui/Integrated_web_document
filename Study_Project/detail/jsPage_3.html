내장객체 및 관련함수


표준 객체(Standard Object) 개요

자바스크립트 언어가 실행되는 어디서나 사용 가능한 객체


1. 문자열(String) 객체


JS에서 문자열은 보통 리터럴 표기법을 사용하여 표현 문자열을 조작할 수 있는 기능 제공


2. 문자열(String) 관련 method


1. indexOf(str)
   인자로 전달받은 문자열(str)이 처음으로 등장하는 위치 반환

2. lastIndexOf(str)
   인자로 전달받은 문자열(str)이 마지막으로 등장하는 위치 반환

3. charAt(인덱스)
   전달받은 인덱스에 위치한 문자 반환

4-1. slice(시작 인덱스, 종료 인덱스)

4-2. substring(시작 인덱스, 종료 인덱스)

4-3. substr(시작 인덱스, 종료 인덱스)
   전달받은 시작 인덱스부터 종료 인덱스 바로 앞까지 문자열을 추출하여 새로운 문자열 반환

5. split("구분자")
   전달받은 구분자를 기준으로 나눈 후, 나뉘어진 문자열을 하나의 배열로 반환

6. concat(str)
   전달받은 문자열을 결합한 새로운 문자열 반환

7. toUppterCase()
   모든 문자를 대문자로 변환한 새로운 문자열 반환

8. toLowerCase()
   모든 문자를 소문자로 변환한 새로운 문자열 반환

9. trim()
   양 끝에 존재하는 모든 공백과 줄 바꿈 문자를 제거한 새로운 문자열 반환


3. 숫자(Number) 객체

JS에서 숫자는 보통 숫자 리터럴을 사용하여 표현


4. 숫자(Number) 리터럴 표기법


1. 정수

2. 실수

3. Infinity(무한)
   양/음의 무한대
4. Nan(Not A Number)
   숫자가 아니라는 의미, 정의되지 않은 값이나 표현할 수 없는 값


숫자(Number) 관련 method


1. Number.parseFloat("문자열")
   문자열에 포함된 숫자 부분을 실수 형태로 반환

2. Number.parseInt("문자열")
   문자열에 포함된 숫자를 정수 형태로 반환

3. Number.isNaN()
   전달된 값이 NaN인지 아닌지 검사 [return : True/False].

4. Number("문자열")
   문자열에 포함된 숫자 부분을 정수면 정수 형태로, 실수면 실수 형태로 반환


수학(Math) 객체

수학에서 자주 사용하는 상수와 함수들을 미리 구현해 놓은 JS 표준 내장 객체


수학(Math) 관련 method


1. Math.min(num1, num2, num3, ...)
   인수로 전달받은 값 중에서 가장 작은 수 반환

2. Math.max(num1, num2, num3, ...)
   인수로 전달받은 값 중에서 가장 큰 수 반환

3. Math.random()
   0보다 크거나 같고 1보다 작은 무작위 숫자 반환

4. Math.floor(num)
   인수로 전달받은 값과 같거나 작은 수 중에서 가장 큰 정수 반환

5. Math.ceil(num)
   인수로 전달받은 값과 같거나 큰 수 중에서 가장 작은 정수 반환

6. Math.trunc(num)
   인수로 전달받은 값의 모든 소수 부분을 삭제하고 정수 부분만 반환


날짜(Date) 객체

시간과 날짜 정보를 담고 있는 객체로 시스템의 현재 날짜, 시간을 조회 및 설정하기 위한 기능 제공


날짜 표현


1. 연도(year)
   1900년(00) ~ 1999년(99)

2. 월(month)
   1월(0) ~ 12월(11)

3. 일(day)
   1일(1) ~ 31일(31)

4. 시(hours)
   0시(0) ~ 23시(23)

5. 분(minutes)
   0분(0) ~ 59분(59)

6. 초(seconds)
   0초(0) ~ 59초(59)


8-2. 날짜 객체 초기화


1. new Date()

2. new Date("날짜를 나타내는 문자열")

3. new Date(ms)

4. new Date(년, 월, 일, 시, 분, 초, ms)


9. 날짜(Date) 객체 method


1. Date.now()
   1970년 1월 1일 0시 0분 0초부터 현재까지 시간을 ms 단위의 정수로 반환

2. Date().getFullYear()
   현재 년도를 4비트의 숫자(YYYY)로 반환

3. Date().setFullYear()
   Date 객체의 값을 특정 날짜로 설정

4. Date().getDay()
   현재 요일에 해당하는 숫자 반환( 일(0)~월(6) )

5. Date().getDate()
   현재 날짜에 해당하는 숫자 반환

6. Date().getDate()
   1970년 1월 1일 0시 0분 0초부터 현재까지 시간을 ms 단위로 환산한 값 숫자로 반환

7. Date().setDate()
   Date 객체의 일자 값을 특정 일자로 설정


배열(Array) 객체

같은 종류의 변수를 하나로 묶기 위한 배열에 관련된 기능 제공


배열(Array) 객체 method


1. Array.isArray()
   전달받은 값이 Array 객체인지 아닌지 검사

2. Array.from()
   배열과 비슷한 객체나 반복할 수 있는 객체를 배열처럼 변환

3. Array.of()
   인수의 수나 타입에 상관없이 인수로 전달받은 값을 가지고 새로운 Array 객체 생성


배열의 특징

자료형과 같이 길이에 제한 없음


1. 배열 선언 방법


1. arr = [배열요소1, 배열요소2, ...]

2. arr = Array[배열요소1, 배열요소2, ...]

3. arr = new Array()

4. arr = [];


2. 배열 요소 추가 방법


1. arr.push(추가할 요소)

2. arr[arr.length] = 추가할 요소

3. arr[특정 인덱스] = 추가할 요소

3. 배열 요소 접근 방법


1. 일반 for문
   인덱스로 접근

2. 향상된 for문
   배열 요소에 바로 접근


4. 배열 method


1. arr.indexOf("값")
   일치하는 값을 가진 요소의 인덱스 반환

2. arr.sort([정렬 기준 함수])
   배열 내 문자열 요소를 오름차순으로 정렬

3. arr.toString()
   배열 요소를 하나의 문자열로 출력

4. arr.join("구분자")
   배열 요소 사이에 구분자를 추가하여 배열을 하나의 문자열로 반환
