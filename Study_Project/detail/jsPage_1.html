JS란

Browser의 동작을 사용자의 요구에 맞게 수행될 수 있도록 하는 언어
소스 코드를 컴파일(compile)하지 않고 인터프리터를 이용하여 소스 코드를 1 line 별로 읽어 바로 실행


1. JS 작성 방법



1. Browser console에 직접 작성

2. 태그에 직접 JS 코드를 작성 : inline
   작성 방법 : <태그 on이벤트명 = "해당 요소에 해당 이벤트 발생 시 실행할 소스 코드">

3. HTML 내부에 < script > 태그를 이용하여 작성 : internal
   HTML 문서 내에 script 태그 영역 안에 소스 코드 작성
   위치는 head, body 태그 전부 작성 가능

4. HTML 외부에 (.js)파일을 이용하여 작성 : external
   별도의 .js 파일을 소스 코드를 작성해서 가져다가 사용하는 방법


2. JS 데이터 입/출력(요소에 대한 정의)



JS 데이터 입력


1. window.confirm("내용")
   질문에 대한 예/아니오 결과를 얻고자 할 때 사용하는 대화 상자 출력 함수
   선택 결과에 따라 확인 버튼 클릭 시 true, 취소 버튼 클릭 시 false 반환

2. window.prompt("내용")
   텍스트를 작성할 수 있는 대화 상자 출력 함수
   작성한 내용을 확인 버튼 클릭 시 true, 취소 버튼 클릭 시 null 반환

3. 선택한 input요소.value
   < input > 태그에 작성된 내용은 "선택한 요소.value" 이용하여 요소의 값 접근


JS 데이터 출력


1. window.alert("내용")
   browser에 대화 상자 띄우는 함수

2. window.console.log("내용")
   browser console창에 띄우는 함수

3. document.write("내용")
   새로운 화면에 내용을 띄우는 함수

4. 선택한 요소.innerText = "요소에 출력할 문구"
   요소에 작성된 내용을 읽거나 변경하는 속성
   내용 변경 시 태그는 문자열로 해석되어 시각적인 요소로 적용되지 않음

5. 선택한 요소.innerHTML = "요소에 출력할 문구"
   요소 전체를 (태그 + 속성 + 내용) 읽거나 변경하는 속성
   내용 변경 시 태그를 해석하여 시각적인 요소로 적용
