함수


1. 함수란


함수는 Javascript에서 기본적인 구성 블록 중의 하나이다.
함수는 작업을 수행하거나 값을 계산하는 문장 집합 같은 JS 절차이다.
함수를 사용하려면 함수를 호출하고자 하는 범위 내에서 함수를 정의해야만 한다.

2. 선언

함수의 이름
괄호 안에서 쉼표로 분리된 함수의 매개변수 목록
중괄호{ } 안에서 함수를 정의하는 자바스크립트 표현


3. 호출

함수를 정의하는 것은 함수를 실행하는 것이 아니다.
함수를 정의하는 것은 간단히 함수의 이름을 지어주고, 함수가 호출될 때 무엇을 할지 지정해주는 것이다.
함수를 호출하는 것은 나타나있는 매개변수를 가지고 지정된 행위를 수행하는 것이다.

                    function 함수명(매개변수){  // 함수 선언

                        // 함수 정의
                
                    }
                
                    함수명();  // 함수 호출


4. 함수의 종류


익명 함수

이름이 없는 함수이기에 필요할 때 호출하는 것이 불가능하고, 이벤트 핸들러와 같이 바로 동작하는 함수가 필요한 경우 또는 변수, 매개변수에 함수를 저장해야 하는 경우 사용

                    function(매개변수){ // 함수 선언
                        // 함수 정의
                    }


즉시 실행 함수

익명 함수의 한 종류로 함수가 정의되자마자 바로 실행되는 함수


즉시 실행 함수를 사용하는 이유
   함수의 선언, 정의, 호출의 일련의 과정을 수행하지 않아도 바로 실행된다.

   사용하려는 변수명이 전역변수로 사용되어지고 있는 경우 즉시 실행 함수를 이용하여 내부의 지역 변수로 작성하여 변수명 충돌 문제를 해결할 수 있다.

                    (function(매개변수){
                        // 함수 정의
                    })();


화살표 함수

익명 함수의 표현식을 간단히 표현한 표기법


1. 작성방법 1

   기본 형태 ([매개변수]) => {함수 정의}

                    익명 함수   : function(){ }
                    화살표 함수 : () => { }

                    익명 함수   : function(num){ return num * 2; }
                    화살표 함수 : (num) => { return num * 2; }

2. 작성방법 2

   매개변수가 "하나"인 경우 () 생략 가능
   매개변수가 없을 경우 무조건 () 작성해야 한다.
                    익명 함수   : function(e) { e.target.style.backgroundColor = "yellow"; }
                    화살표 함수 : e => { e.target.style.backgroundColor = "yellow"; }

3. 작성방법 3

   함수 정의 부분이 return [ 식 또는 값 ]으로만 작성되어 있는 경우 {}, return 생략 가능
   return 되는 값이 객체(object)인 경우 {}, return 생략 불가능하다.
                    // 함수 호출(익명 함수)
                      function(num){ return num * 10 };
                    // 함수 호출(화살표 함수)
                      num => num * 10;

4. 화살표 함수 주의 사항

   화살표 함수에서는 this 사용이 불가능하다.
   화살표 함수에서 this는 창 자체를 나타내는 객체(window)를 의미한다.
