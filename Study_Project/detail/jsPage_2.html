JS 요소


요소 접근 방법



1. DOM을 이용한 HTML 요소 접근하기

DOM(Document Object Model) : 웹 문서(HTML)의 모든 요소를 객체 형식으로 표현한 것
해당하는 요소 객체를 가져오는 것을 의미


1. id로 접근하기
  document.getElementById("id 속성 값");

2. name으로 접근하기
  document.getElementByName("name 속성 값");

3. class로 접근하기
  document.getElementByClassName("class 속성 값");

4. tag로 접근하기
  document.getElementByTagName("태그 명");

5. CSS 선택자로 접근하기
  document.querySelector("CSS 선택자");   document.querySelectorAll("CSS 선택자");


• 주의 사항
- 접근한 요수가 여러 개일 경우(class, name, tag, querySeclectorAll)에는 복수 형태(Elements)로 작성
- 요소를 여러 개 접근하는 경우 [배열] 형태로 반환된다.
- 요소의 값을 얻어올 때 -> 인덱스를 이용해서 요소 하나씩 접근한다.


2. input 태그의 값(value) 얻어오기 / 변경하기


1. input 태그는 [value]를 이용해서 값을 얻어 오거나, 변경할 수 있다.
  value를 사용해서 input 태그에 입력 받은 값을 String 타입으로 얻어올 수 있다.
  value 없이 얻어온다면 값의 타입은 Object가 된다.

2. input 태그의 주어진 속성 값으로 요소에 접근 후 값을 얻어온다.

3. 추가 정보 & 관련 함수
  input의 값을 얻어온 후 작성된 값 초기화하기
  input에 초점 맞추기
  입력 받은 문자열 양 끝 공백 제거하기


변수와 자료형



1. 변수 선언 위치에 따른 구분

JS에서 변수는 전역변수와 지역변수로 구분한다.


1. 전역변수: 선언된 후 같은 문서 내 어디서든 사용 가능
  선언 방법 : 함수 외부 및 함수 내부에 선언 시 변수명 앞에 아무런 키워드가 없으면 전역변수

2. 지역변수 : 함수 내부에 선언되는 변수로 해당 함수 내에서만 사용 가능
  선언 방법 : 함수 내부에 키워드 + 변수명으로 작성


2. 변수 선언 방식


1. var : 변수, 변수명 중복 O(덮어쓰기), 함수 레벨 scope
  (ES6 이후부터 사용 빈도가 많이 적어짐)

2. let : 변수, 변수명 중복 X, 블록{} 레벨 scope

3. const : 상수, 변수명 중복 X, 블록{} 레벨 scope


3. JS 자료형

JS는 변수 선언 시 별도의 자료형을 지정하지 않고, 변수에 대입되는 값에 의해서 자료형 결정


4. JS 자료형 종류


  1. String [ 문자열 ]
  2. Number [ 정수, 실수 ]
  3. Boolean [ 논리값 ]
  4. Object [ 객체 {K:V} & 배열 ]
  5. Function [ 함수 ]
  6. Undefined [ 정의되지 않은 변수 ]
  7. Null [ 참조하는 주소 없음 ]
