DOM


1. DOM이란

HTML 문서를 객체 기반으로 표현한 것으로 HTML 문서에 작성된 내용을 TREE 구조로 나타냈을 때 각 TAG, TEXT, COMMENT 등을 Node라고 부른다.



2. Node와 Element


Node

TAG(요소 노드), 속성, 주석, 내용(TEXT 노드) 등 모두 표현


Node 탐색 방법


1. 부모 노드 탐색
   .parentNode

2. 새로운 노드 추가
   .append()

3. 첫 번째 자식 노드 탐색
   .firstChild

4. 마지막 자식 노드 탐색
   .lastChild

5. 중간에 존재하는 자식 노드 탐색 부모 요소들을
   .childNodes[인덱스]
6. 이전 형제 노드 탐색
   .previousSibling
7. 다음 형제 노드 탐색
   .nextSibling


Element

Node의 하위 개념으로 요소 노드(TAG) 만을 표현

Element만 탐색하는 방법


1. 자식 요소만 모두 선택
   .children

2. 부모 요소 선택
   .parentElement

3. 첫 번째 자식 요소 선택
   .firstElementChild

4. 마지막 자식 요소 선택
   .lastElementChild

5. 이전 형제 요소 선택
   .previousElementSibling

6. 다음 형제 요소 선택
   .nextElementSibling

4. 요소 추가 제거


1. 추가 제거 관련 함수


1. 요소 추가
   요소.add()

2. 요소 제거
   요소.remove()

3. 요소에 class 추가
   요소.classList.add("class명")

4. 해당 요소에 적용된 class 제거
   요소.classList.remove("class명");

5. 요소에 속성 / 속성 값 추가
   요소.setAttribute("속성", "속성 값")


2. 추가 제거 Example


                  // 추가 버튼(#add)가 클릭 되었을 때
                  document.getElementById("add").addEventListener("click", function(){

                  // div 요소 생성
                  const div = document.createElement("div");
                  
                  // div에 row 클래스 추가
                  div.classList.add("row");
                  
                  // --------------------------------------------------
                  
                  // input 요소 생성
                  const input = document.createElement("input");
                  
                  // input에 in 클래스 추가
                  input.classList.add("in");
                  
                  // input에 "type" 속성, "number" 속성값 추가 (type="number")
                  
                  // - 요소.setAttribute("속성", "속성값") : 요소에 속성/속성값 추가
                  input.setAttribute("type", "number");
                  
                  // -------------------------------------------------
                  
                  // span 요소 생성
                  const span = document.createElement("span");
                  
                  // span에 remove 클래스 추가
                  span.classList.add("remove");
                  
                  // span의 내용으로 "X" 추가
                  span.innerText = "X";
                  
                  // span이 클릭 되었을 때에 대한 이벤트 동작 추가
                  span.addEventListener("click", function(){
                  
                      // 요소.parentElement : 부모요소
                      // 요소.remove() : 요소 제거
                  
                      // 부모(.row) 제거
                      span.parentElement.remove();
                  
                  })
                  
                  
                  // ----------------------------------------------------
                  
                  
                  // div 내부에(자식으로) input, span 순서대로 추가
                  div.append(input);
                  div.append(span);
                  
                  // #container에 div를 마지막 자식으로 추가
                  document.getElementById("container").append(div);

              });
